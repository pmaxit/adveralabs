## Design System Notes

### Color Palette
- Primary: #6366F1 (Indigo) - Trust, professionalism
- Secondary: #8B5CF6 (Purple) - Innovation, creativity
- Accent: #EC4899 (Pink) - Energy, action
- Success: #10B981 (Green) - Growth, positive results

### Typography
- Font: Inter (modern, readable, professional)
- Headings: 700-800 weight for impact
- Body: 400-500 weight for readability

### Spacing
- Consistent 8px grid system
- Generous whitespace for breathing room
- Section padding: 80px (desktop), 48px (mobile)

### Components
- Rounded corners: 8-12px for modern feel
- Shadows: Subtle elevation hierarchy
- Gradients: Used sparingly for emphasis
