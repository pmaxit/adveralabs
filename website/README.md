# Advera Labs Website

Modern, conversion-optimized website for Advera Labs - AI-powered cross-channel ad optimization platform.

## Features

- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **Modern Aesthetics**: Clean design with gradient accents and smooth animations
- **Conversion-Focused**: Multiple CTAs, ROI calculator, social proof
- **Competitive Differentiation**: Clear comparison table vs competitors
- **Performance Optimized**: Lightweight, fast-loading

## Structure

```
website/
├── index.html      # Main HTML file
├── styles.css      # All styles and responsive design
├── script.js       # Interactive features and animations
└── README.md       # This file
```

## Key Sections

1. **Hero**: Value proposition with dashboard preview
2. **Problem**: Pain points current tools don't solve
3. **Solution**: Feature showcase (4 pillars)
4. **How It Works**: 4-step process
5. **Comparison**: vs competitors table
6. **ROI Calculator**: Interactive savings calculator
7. **Integrations**: Platform logos
8. **Social Proof**: Testimonials
9. **Pricing**: Three-tier pricing
10. **CTA**: Final conversion section

## Design System

### Colors
- Primary: #6366F1 (Indigo)
- Secondary: #8B5CF6 (Purple)
- Accent: #EC4899 (Pink)
- Success: #10B981 (Green)

### Typography
- Font: Inter (Google Fonts)
- Headings: 700-800 weight
- Body: 400-500 weight

### Spacing
- 8px grid system
- Section padding: 80px (desktop), 48px (mobile)

## Getting Started

1. Open `index.html` in a web browser
2. No build process required - pure HTML/CSS/JS
3. For production, minify CSS/JS and optimize images

## Customization

### Update Content
- Edit `index.html` for text content
- Modify `styles.css` for colors/spacing
- Add functionality in `script.js`

### Add Images
- Replace SVG placeholders with actual logos
- Add customer logos to integrations section
- Add dashboard screenshots to hero
